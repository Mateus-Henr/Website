Do you want to know more?

Java is the computational environment, or platform, created by the US company Sun Microsystems, and sold to Oracle after a few years. The platform allows developing programs using the language of Java programming.

Just like Windows, iOS, and Mac OS, Linux is an operating system. In fact, one of the most popular platforms on the planet, Android, is powered by the Linux operating system. An operating system is software that manages all of the hardware resources associated with your desktop or laptop. To put it simply, the operating system manages the communication between your software and your hardware.

Clojure is a dynamic, general-purpose programming language supporting interactive development. Clojure is a functional programming language featuring a rich set of immutable, persistent data structures. As a dialect of Lisp, it has a code-as-data philosophy and a powerful macro system.

MySQL is a database management system that uses an SQL language as an interface. It is currently one of Oracle Corporation's most popular database management systems, with over 10 million installations worldwide.

Why choose us

  • Clojure

    Our team chooses to use Clojure due to its use in a variety of projects, from the web to database projects, its implementation is done through JAVA which is a known language.

  • Scrum

    We use this framework known as Scrum as a strategy to accelerate our processes. From the simplest to the most difficult, maintaining an organization and quality at the end of each project.

  • NodeJS

    We use Node.js because it is possible to create Javascript applications to run as a standalone application on a machine, not depending on a browser to run, as we are used to.

  • PostgreSQL

    We are adept at PostgreSQL because it is one of the most advanced open-source database management systems on the market. There are several ways of extensibility such, languages, functions and operators.
